--------------------------------------------------------------------------
R U N N I N G
--------------------------------------------------------------------------
System Requirements:
Java version 1.5 or newer.
Globus Toolkit Core 1.4.2

Apache Ant version 1.6 or newer is required for a source build only.

--------------------------------------------------------------------------
B U I L D I N G
--------------------------------------------------------------------------
You must have ant version 1.6 or newer installed.
To build type: ant all
For information on available targets, type: ant -projecthelp

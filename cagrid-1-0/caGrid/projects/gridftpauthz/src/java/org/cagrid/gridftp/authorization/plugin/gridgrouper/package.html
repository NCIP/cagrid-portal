Grid Grouper Authorization Plug-in

Package Specification

	This package contains the Grid Grouper Authorization Plug-in.
	
	A user wanting to learn more about the plug-in should start by reading the
documentation for {@link org.cagrid.gridftp.authorization.plugin.gridgrouper.GridGrouperAuthCallout}.

Related Documentation

	For more information on using the plug-in refer to the GridFTP with Java Authorization
User Documentation packaged with the API distribution.

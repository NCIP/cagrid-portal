Database Authorization Plug-in

Package Specification

	This package contains the Database Authorization Plug-in.
	
	A user wanting to learn more about the plugin should start by reading the
documentation for {@link org.cagrid.gridftp.authorization.plugin.db.DatabaseAuthCallout}.

Related Documentation

	For more information on using the plug-in refer to the GridFTP with Java Authorization
User Documentation packaged with the API distribution.
